The changed files app is an optional add-on for Watchful, the site maintenance toolbox for WordPress. The app allows you to monitor changes for up to 10 files on your wordpress site(s).

== Description ==

Watchful is a web developers toolbox for remotely managing and monitoring multiple WordPress websites. It automatically monitors the critical files on your website.

By installing the Changed Files app, up to 10 files of your choosing may be monitored as well.

Any changes detected in the specified files will be logged. Notification emails may also be sent as desired.

== Requirements ==

Your website must already have the main [Watchful plugin](https://wordpress.org/plugins/watchful/) installed. The site must also be added to your Watchful account.

== Installation ==

1. Install and activate the Changed Files plugin through the 'Plugins' menu in WordPress.
2. On the post-install screen, click the button called 'Add to Watchful'.
